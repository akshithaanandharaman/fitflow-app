# 💪 FitFlow – Premium Workout Tracker App

FitFlow is a **modern, high-performance mobile workout tracking application** built using **React Native (Expo)** and enhanced with **GSAP animations**.  
It delivers a **premium fitness experience** with smooth interactions, visually rich UI, and real-time workout tracking.

---

## 🚀 Overview

FitFlow helps users track workouts, monitor progress, and stay consistent with their fitness goals through an intuitive and engaging interface.

The app focuses on:
- ⚡ Performance
- 🎨 Premium UI/UX
- 🎬 Advanced animations
- 📊 Data visualization

---

## ✨ Features

### 🔐 Authentication
- Clean Login & Signup UI
- Animated onboarding experience

### 🏠 Dashboard
- Daily workout summary
- Calories burned & time tracked
- Animated progress indicators

### 🏋️ Workout Tracker
- Add, edit, delete exercises
- Track reps, sets, and duration
- Predefined workout categories (Chest, Legs, Cardio, etc.)

### ⏱️ Smart Timer
- Built-in workout timer
- Animated countdown with smooth transitions

### 📊 Analytics
- Weekly & monthly performance insights
- Animated charts (bar & line)

### 🎯 Goal Tracking
- Set and monitor fitness goals
- Dynamic progress visualization

### 🔔 Reminders (UI)
- Smart notification cards with subtle animations

---

## 🎨 UI/UX Design

- 🌙 Dark theme (premium look & feel)

- 🎯 Clean typography & spacing

## 🎬 Animations (Powered by GSAP)
